All my ECUs run the latest version of the MS Extra code. (Well, they would do, as I'm one of the authors of the MS-Extra code and author of the MS1-Extra and MS2-Extra manuals. I'm also an administrator of the MSExtra forum.)

MegaSquirt is a standalone DIY mappable Engine Management System that can control the fuel and spark or fuel only or spark only on virtually any liquid fueled spark-ignition engine, naturally aspirated or boosted.

Are you still confused about MegaSquirt? Click here.

A MegaSquirt ECU can give you mappable fuel and or mappable ignition using:

  • A distributor setup with a simple hall sensor/points in the distributor or a cam/crank signal.
  • A coil pack setup using a variety of cam/crank signals.
  • A Coil On Plug setup using a variety of cam/crank signals.

All of the ECU's I sell have the MS-Extra code loaded into them, these are the features that come as standard:

  • # 16x16 Fuel table
  • # 12x12 Spark table
  • # Rev limiter
  • # Overboost protection
  • # Mappable Air Fuel Ratio targets
  • # Wide Band or Narrow Band
  • # Coolant temperature related ignition timing advance
  • # Inlet Air temperature related ignition timing retard
  • # Manifold Absolute Pressure rate of change or Throttle Position Sensor rate of change acceleration enrichment
  • # Overrun fuel cut
  • # Hybrid Alpha-N
  • # Staged Injection
  • # Built-in Air Density Correction
  • # PWM Idle Control valves
  • # Stepper Motor Idle Control Valves
  • # Fuel and Ignition or Fuel Only or Ignition Only
