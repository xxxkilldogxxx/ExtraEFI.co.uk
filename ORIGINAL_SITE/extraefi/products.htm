What Do I Get as Standard?

Still Confused about MegaSquirt's?

All the ECU packages I sell include the following as standard:

One year, return to base, warranty see here for details.

 (MS1, MS2 and MS3 Packages):
A wiring loom (approx. 1 meter long) with a high audio quality screened cable for the trigger signal, a 0.75-1.0mm ignition supply cable and multiple ground cables, to ensure the ECU gets a good electrical connection helping to eliminate potential noise!

(MS1, MS2 and MS3 packages):
The supplied loom wires are bare ended ready for you to wire to your setups connectors. Each wire has a label with its description on it and they are all colour coded for making wiring easier, this all corresponds to the wiring diagram on the free CD.

(MS1, MS2 and MS3 packages):
ECU's that have more than 2 Spark Coil Outputs have a seperate DB15 connector added to the front of the ECU specifically for supplying the Spark Coil Outputs, this ensures that the spark driver circuits earthing point and connectors are more than capable of the extra current needed for driving multiple coils. The connector has a separate loom, allowing complete separation and ease of installation. There is also a seperate ground for the spark drivers in this loom, ensureing noise is kept away from the ECU's sensitive grounds. If using the MS3X card then that has seperate 37 pin connector for the low voltage spark drivers.

(MS1, MS2 and MS3 packages):
All ECUs with more than 4 spark drivers (e.g. V10 and V12 wasted) have an externally mounted alluminium box with the spark drivers inside. This ensures the noise from the drivers is kept well away from the ECU. See HERE for a picture. If using the MS3X card then that has seperate 37 pin connector for the low voltage spark drivers.

Spark Drivers:
As of May 2008 all Spark drivers are a Bosch component (BIP132). To date, this item has been 100% reliable!!

High Powered Idle Valve Output Transistor:
Capable of supplying more than 3 Amps, giving it the ability to directly drive idle control valves, such as the 3 or 2 wired Bosch units or the Ford items, etc. This means you do NOT need to purchase a Relay Board for a standard install!! The original MS idle output transistor simply isn't up to the job of controlling anything other than a relay. The ECU's I supply can directly control the idle valve! See here for more information.

I endeavor to set up a base configuration file for your engine from the information you let me have, this helps you get running a lot quicker

I will setup the ECU and include the files needed to use your standard equipment temperature sensors if you can tell me some resistance values from the sensors.

A communications lead (RS232) for connection to your computer/laptop. (Newer laptops may need an RS232-USB adaptor if they dont have the RS232 port)

A free CD (download it here if you wish) with:

* MS-Extra Software/Hardware Manuals
* Tuner Studio - The latest tuning software
* MSLVV (Datalog viewing software)
* EasyTherm (temperature sensor configuration software)
* Drawing of suggested layouts corresponding to the wiring loom
* Instructions for software installation, temperature sensor help files
* The relevant firmware, etc.

All of the ECU's I sell have the MS-Extra code loaded into them, this has these features as standard:

# 16x16 Fuel Tables (MS2-Extra, MS3 and all PnP ECUs) (Manifold Absolute Pressure x RPM or Throttle Position x RPM)

# 12x12 Fuel Tables (MS1-Extra) (Manifold Absolute Pressure x RPM or Throttle Position x RPM for ITB's, etc)

# 12x12 Spark Tables (MS3 has 16x16) (Manifold Absolute Pressure x RPM or Throttle Position x RPM for ITB's, etc.)

# Rev Limiter (Retards ignition then round robin spark cut or fuel cut or both)

# Overboost Protection (Round robin spark cut or fuel cut or both)

# 12x12 Mappable Air:Fuel Ratio Targets (MS2-Extra/MS3/PnP) (For use with a wideband lambda Sensor)

# 8x8 Mappable Air:Fuel Ratio Targets (MS1-Extra) (For use with a wideband lambda Sensor)

# WideBand or NarrowBand Lambda Sensor compatability

# Coolant Temperature related Ignition Timing Advance (helps speed up idle during warmup period)

# Inlet Air Temperature related Ignition Timing Retard (helps prevent detonation due to excess heat soak)

# Manifold Absolute Pressure rate of Change or Throttle Position Sensor rate of Change Acceleration Enrichment

# Overrun fuel cut (Reduces fuel usage by cutting fuel during overrun)

# Hybrid Alpha-N (For very lumpy cams, ITB's, massive throttle bodies, running boost.)

# Staged Injection (Ideal for 2 different sized injectors e.g. Rotary's)
# Built-in Air Density Correction (Using Manifold Air Temperature Sensor)

# PWM Idle Control valves (For use with 2 or 3 wired variable valves)

# Stepper Motor Idle Control Valves (MS2-Extra/MS3/PnP
Note: MS1-Extra can NOT control STEPPER MOTORS!)

# Fuel and Ignition or Fuel Only or Ignition Only (Can be used for a fuel only controller or ignition only or both)

All ECU's are built by myself (a fully qualified Electronics Engineer with over 30 years experience), this ensures I can maintain a very high quality. I've built something like 600+ MS ECU's to date!
Each ECU is tested on my unique test bed, the same one the MS-Extra code was developed on, under various simulated engine conditions. Next, each ECU is soak tested (run continuously) under load for a minimum of 6Hrs. The ECU is then tested again before it is sprayed with a conformal coating to protect it.
